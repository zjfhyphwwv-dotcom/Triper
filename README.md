# Triper

Небольшое консольное приложение со списком самых популярных направлений
на Камчатке. Используйте модуль `src/destinations.py`, чтобы получить
структурированную информацию о точках притяжения и их активностях.

## Запуск

```bash
python src/destinations.py
```

Скрипт выведет названия направлений, краткие описания и список популярных
активностей, чтобы помочь спланировать путешествие по полуострову.
