# Triper

Простое прототипное приложение для отображения популярных направлений на Камчатке.

## Данные

Данные расположены в файле [`app/data/kamchatka_destinations.json`](app/data/kamchatka_destinations.json) и содержат:

- название направления;
- категорию и краткое описание;
- ключевые впечатления для путешественника;
- рекомендованный сезон для поездки;
- географические координаты.

## Использование

```bash
python -m app.cli            # выводит JSON со списком направлений
python -m app.cli "Толбачик"  # подробности по конкретному месту
```
